## XEntity Inventory Toolkit
### An inventory and item system toolkit developed for unity. This tool primarily focuses on speeding up ealry development with a modular design for easy integration.

### Features
- Item container system with customizable UI.
- Customizable item definition system.
- Customizable item pickup and utilization logic. 
- Inventory presistance: easily write and read inventory data from disk.

### Try it on your project
#### Clone this repository into your assets folder or install directly from: https://assetstore.unity.com/packages/tools/game-toolkits/xentity-inventory-item-system-205357
